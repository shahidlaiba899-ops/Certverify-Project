<img width="1600" height="781" alt="image" src="https://github.com/user-attachments/assets/e153d005-a53a-404a-a296-d3c0739f9c72" /># 🎓 CertVerify — Certificate Verification System

A modern web-based certificate verification system built using HTML, CSS, and JavaScript. This project allows users to verify digital certificates instantly using a unique Certificate ID.

---

## 🌐 Live Demo

https://digital-certificate-verifier.vercel.app

---

## 🚀 Features

* 🔍 Single Certificate Verification
* 📂 Bulk Verification System
* 📜 Dynamic Certificate Generation
* 🖨 Print / Save as PDF
* 🎨 Professional UI Design
* ⚡ Fast & Lightweight (No backend required)

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
Certverify-Project/
│── index.html          # Main verification page
│── certificate.html   # Certificate display page
│── style.css          # Styling file
```

---

## ▶️ How to Use

1. Open the Live Demo link
2. Enter Certificate ID (e.g., CV-1001)
3. Click on "Verify"
4. View certificate details instantly
